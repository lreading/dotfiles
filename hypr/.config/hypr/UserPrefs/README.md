- `colors-waybar.css`: pinned Waybar wallust colors.
- `kitty.conf`: pinned Kitty config.
- `kitty-01-Wallust.conf`: pinned Kitty wallust theme.

- Rainbow border re-application after the delayed Hyprland reload in `WallustSwww.sh`
